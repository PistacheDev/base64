# Base64
Base64 Encoding and Decoding

Base64 Preview: [Youtube](https://youtu.be/yFJTNGeiluY)

# How To Use
- Type `encode -TYPE ANYTHING YOU LIKE-` for encoding
- Type `decode -TYPE ANYTHING YOU LIKE-` for decoding
